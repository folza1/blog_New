Laravel 8 From Scratch Blog Project from Laracasts

**Terminálban:**

cd elérési_út/app

cp .env.example .env

composer install

php artisan key:generate

php artisan serve

Npm install

Npm run dev

**Adatbázis beállítása**

php artisan migrate


**Symlink beállítása**

php artisan storage:link

.env fájlban:

FILESYSTEM_DRIVER=public
